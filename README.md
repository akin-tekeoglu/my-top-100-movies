## About

An app something similar to [this](https://www.themoviedb.org/). Users can make their own list and interact with each other's list.

## Requirements

- node 14
- npm 7
- An API Key from [here](https://www.themoviedb.org/)


## Installation

- `npm install`
- `echo API_KEY=your-api-key > .env`
- `npm run start:dev`
